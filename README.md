# 🤝 Site de Voluntariado

Um site moderno e inspirador para recrutar e engajar voluntários em causas comunitárias. Desenvolvido com HTML semântico, CSS responsivo e JavaScript interativo.

## 📋 Características

- **Hero Section Impacto**: Seção inicial com design colorido e animado
- **Sobre o Projeto**: Informações sobre a organização e impacto
- **Formas de Ajudar**: 4 cards descritivos com iconografia
- **Formulário de Inscrição**: Validação completa de dados com feedback visual
- **Design Responsivo**: Adapta-se a diferentes tamanhos de tela
- **Interatividade**: Scroll suave, animações CSS, validação em tempo real

## 🎨 Paleta de Cores

- **Primária**: Azul vibrante (#0066FF) - Confiança e ação
- **Secundária**: Coral (#FF6B6B) - Energia e paixão
- **Neutros**: Branco, cinzas e preto para boa legibilidade
- **Destaque**: Amarelo (#FFB700) para CTAs

## 🛠️ Tecnologia

- **HTML5** - Estrutura semântica
- **CSS3** - Gradientes, flexbox, grid e animações
- **JavaScript** - Validação de formulário e interatividade

## 📁 Estrutura de Arquivos

```
.
├── index.html      # Estrutura HTML com 6 seções
├── style.css       # Estilos CSS com paleta colorida
├── script.js       # Validação e interatividade
└── README.md       # Este arquivo
```

## 🚀 Como Usar

1. Abra `index.html` em um navegador web
2. Navegue pelas seções usando o menu no header
3. Preencha o formulário de inscrição com:
   - Nome completo (obrigatório)
   - Email válido (obrigatório)
   - Pelo menos uma área de interesse (obrigatório)
4. Clique em "Confirmar Inscrição"

## ✅ Funcionalidades Implementadas

### Validação de Formulário
- ✔️ Campo de nome (mínimo 3 caracteres)
- ✔️ Campo de email (validação de formato)
- ✔️ Áreas de interesse (mínimo 1 seleção)
- ✔️ Mensagens de erro em tempo real
- ✔️ Mensagem de sucesso com nome do voluntário

### Interatividade
- ✔️ Scroll suave entre seções (clique no menu)
- ✔️ Animações de fade-in em cards
- ✔️ Hover effects em botões e cards
- ✔️ Efeito de shadow dinâmico no header

### Design & UX
- ✔️ Gradientes coloridos (hero, formas de ajudar)
- ✔️ Ícones emoji para diferentes áreas
- ✔️ Layout responsivo (desktop, tablet, mobile)
- ✔️ Acessibilidade básica (labels, contrast, focus states)

## 📱 Responsividade

- **Desktop**: Layout completo com 4 cards em grid
- **Tablet**: Grid adaptado para 2 colunas
- **Mobile**: Layout em coluna única com menu compacto

## 🔄 Fluxo do Usuário

1. Chega pela hero section com CTA "Quero Voluntariar"
2. Lê sobre o projeto (stats e descrição)
3. Explora 4 formas de ajudar
4. Preenche formulário de inscrição
5. Recebe confirmação visual
